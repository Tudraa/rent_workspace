# Workspace Designer

## Project Description

Workspace Designer is an interactive web application that allows users to design and customize their ideal workspace. Users can select from a variety of core furniture items such as desks, chairs, monitors, and accessories, and also add various "lifestyle" items like coffee stations or outdoor gear. The application provides an intuitive drag-and-drop interface for arranging items, real-time price calculation, and a streamlined checkout process.

## Features

- **Interactive Workspace Canvas**: Drag and drop furniture and accessories to arrange your ideal setup.
- **Categorized Product Selection**: Browse items by categories such as Chairs, Desks, Monitors, Accessories, Coffee Station, Outdoor Gear, Relax Zone, and Garage Space.
- **Real-time Price Calculation**: See the total monthly cost of your selected items update instantly.
- **Checkout Process**: Review your order summary and confirm your workspace setup.
- **Modern UI/UX**: Built with a clean and responsive design.

## Getting Started

To get a copy of the project up and running on your local machine for development and testing purposes, follow these steps.

### Prerequisites

Make sure you have Node.js (version 18 or higher) and npm installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Tudraa/rent_workspace.git # Assuming this is the repo URL
    cd another_monis_workspace
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

To run the application in development mode:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Live Demo

You can access a live demo of the Workspace Designer application here: [https://rent-workspace-mw2i0j1js-indra-mahendras-projects.vercel.app/](https://rent-workspace-mw2i0j1js-indra-mahendras-projects.vercel.app/)

## Building for Production

To build the application for production:

```bash
npm run build
```

Then, to start the production server:

```bash
npm run start
```
